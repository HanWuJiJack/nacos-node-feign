# Node.js TypeScript Module

## 功能描述
提供问候语生成功能的Node.js模块，使用TypeScript开发并编译为ES模块。

## 安装方式
```bash
npm install node-ts-package
```

## 使用示例
```javascript
import { greet } from 'node-ts-package';

console.log(greet('Developer'));  // Hello Developer, this is Node.js module!
```
